# 📊 HR Analytics Dashboard – Employee Attrition Analysis

Project Overview

This project focuses on analyzing employee attrition using HR data to identify key factors affecting employee turnover. The dashboard provides meaningful insights that help organizations understand why employees leave and how retention can be improved.

---

Objective

* To analyze employee attrition patterns
* To identify key factors influencing attrition
* To provide actionable insights for better decision-making

---

Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning & Transformation
* Data Visualization Techniques

---

Key Features of Dashboard

* KPI Cards (Total Employees, Attrition Count, Attrition Rate, Average Income)
* Attrition by Department
* Attrition by Job Role
* Attrition by Age Group
* Attrition by Salary (Income Groups)
* Work-Life Balance Analysis
* Overtime Impact on Attrition
* Interactive Filters (Slicers)

---

Key Insights

* Higher attrition observed among younger employees
* Employees with lower income show higher attrition rates
* Overtime has a strong impact on employee turnover
* Certain job roles (e.g., Sales) have higher attrition
* Poor work-life balance contributes to employee exits

---

Conclusion

The dashboard highlights critical areas where organizations can improve employee retention, such as better compensation, improved work-life balance, and optimized workload management.

---

Future Improvements

* Integration with Power BI for advanced analytics
* Predictive modeling for attrition forecasting
* Real-time data updates
